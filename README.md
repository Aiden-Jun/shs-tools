# shs-tools
Tools for [stevenson.space](stevenson.space) maintenance written in Python

## lunch_maker.py
Run the file, enter the year and the month.
The tool automatically finds the school days for those days, and only asks the menus for those dates (Doesn't count no school week days).
Enter weekly menus once, then enter daily menus. At any time, type "back" to redo the previous inputs if a mistake is made.
In the end, it will copy to your clipboard, or save as a file.
